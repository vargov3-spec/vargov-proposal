# Vargov®Design — дизайн-система коммерческого предложения

Единая система для PDF и веб-приложения. Референс — премиальный европейский
каталог (Apple / B&B Italia / Louis Vuitton): тишина, воздух, дисциплина.

## Цвета — только фирменные

Палитра сведена к бренду Vargov®Design: чёрное на бумаге и **единственный**
акцент — золото из логотипа. Никаких других оттенков (красный, синий и т.п.
запрещены — даже для ошибок и статусов).

| Токен | Hex | Роль |
| --- | --- | --- |
| Obsidian / Ink | `#1a1a1a` | Основной текст, заголовки, QR, тёмные плашки |
| Graphite | `#454545` | Вторичный текст, абзацы |
| Grey | `#8a8a8a` | Подписи, метки, метаданные |
| Hairline | `#dcdcdc` | Разделители-волоски, границы |
| Faint | `#f7f6f4` | Фон блоков (оплата, примечания) |
| Paper | `#ffffff` | Холст страницы |
| **Gold** | **`#bf8219`** | Единственный акцент: индекс раздела, одна линия, play, левый маркер |

Серые — это тональные производные чёрного (не самостоятельные «цвета»), нужны
для иерархии. Золото появляется **один раз на разворот** как смысловой акцент.

## Типографика

Montserrat (встроен как TTF, текст остаётся живым и векторным).

- **Display** — Light, 26–52 pt: обложка (артикул), заголовки страниц.
- **Body** — Regular/Light, 9.5–15 pt, межстрочный воздух (lineGap 4–7).
- **Emphasis** — Medium/SemiBold: суммы, «ИТОГО», названия.
- **Eyebrow / Label** — Medium 7–8 pt, ВЕРХНИЙ РЕГИСТР, трекинг 1.8–3.4.

Дисциплина: крупный контраст размеров, без «средних» промежуточных заголовков.

## Раскладка

- A4, внешние поля 52 pt, широкие внутренние отступы.
- **Разделение — только волосяной линией** (`#dcdcdc`). Без теней, без рамок-карточек с заливкой (кроме мягкого `#f7f6f4` для блоков оплаты/примечаний).
- Геометрия острая; скругления — минимальны (2–3 pt только у веб-элементов).
- Один акцент на страницу: золотой индекс раздела ИЛИ короткая золотая линия ИЛИ золотой play/маркер.

## Компоненты (PDF)

- **Обложка** — золотой логотип по центру, крупное фото, артикул Light.
- **Eyebrow** — золотой номер `01` + серая метка в разрядку.
- **Строка-факт** — метка слева (grey), значение справа (ink), волосок снизу.
- **Суммы** — USD жирно, под ним рублёвый эквивалент grey (внутренний курс).
- **Плашка оплаты** — фон `#f7f6f4`, золотая полоса слева 2.5 pt.

## Do / Don't

**Do**
- Держи максимум воздуха; пустота — часть композиции.
- Разделяй волосками, не рамками и не тенями.
- Используй золото ровно один раз на странице.
- Держи ≤ 3 уровней серого в тексте (ink / graphite / grey).

**Don't**
- Не вводи посторонние оттенки (красный/синий/зелёный) — палитра только бренд.
- Не добавляй тени, свечения, градиенты.
- Не делай «средних» заголовков — разрыв масштабов и есть система.
- Не заливай карточки цветом; фон блока — только `#f7f6f4`.
